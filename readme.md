## Deployment

Para correr este codigo

```bash
  npm run dev
```

Crear Base de datos en postgreSQL, revisar la carpeta SQL para la estructura

Crear archivo .env con los datos de conexion (el llamado del `env` esta creado con `loadEnvFile`).
